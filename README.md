# MyPortfolio

Professional personal portfolio website showcasing projects, skills, education and services.

## Features

- Responsive, modern UI with Tailwind + custom CSS
- Animated sections and scroll-triggered reveals
- Project, Experience, Education and Services pages
- Reusable page template

## Files

- [index.html](index.html) — Landing / home
- [services.html](services.html) — Services & process
- [skills.html](skills.html) — Skills, certifications
- [education.html](education.html) — Education timeline
- [experience.html](experience.html) — Work experience & projects
- [template.html](template.html) — Base template for new pages
- [assets/css/style.css](assets/css/style.css) — Custom styles
- [assets/js/main.js](assets/js/main.js) — Page scripts
- [assets/images/](assets/images/) — Images used by the site

## Local preview

Open a simple static server in the project root:

```sh
# Python 3
python -m http.server 8000
# or using npm package
npx http-server -p 8000
```
